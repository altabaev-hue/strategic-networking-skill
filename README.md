# Strategic Networking Skill

> Агент разведки с экосистемой контактов — скилл для Cowork / Claude

Скилл превращает Claude в персонального агента стратегического нетворкинга.
Работает поверх двух markdown-файлов которые ты хранишь локально.

## Концепция

Нетворкинг — это система, не случайность.  
Логика: **Целеполагание → Анализ сети → Стратегия → Воронка → Углубление**

Каждый контакт хранится в двух слоях:
- **Стратегический** — зачем этот человек в твоей сети
- **Человеческий** — как с ним работать (стиль, интересы, поводы для касания)

## Что умеет

| Команда | Что делает |
|---|---|
| `добавь / обнови [имя]` | Обновляет профиль после встречи |
| `покажи мою сеть` | Обзор по кругам + анализ пробелов |
| `кто поможет с [задачей]` | Поиск нужного человека |
| `кого давно не трогал` | Флаг замороженных + готовые поводы |
| `готовлюсь к встрече с [имя]` | Полный брифинг перед встречей |
| `что подарить / как написать [имя]` | Уместный канал, тон, текст |
| `weekly review` | Замороженные, ДР, 3 касания на неделю |

## Три круга контактов

- 🔴 **Развитие** — клиенты, инвесторы, партнёры (прямой путь к целям)
- 🟡 **Продуктивность** — эксперты, коннекторы, открывающие двери
- 🔵 **Поддержка** — менторы, доверенные советники, личный круг

## Роли в сети

- **Коннектор** — соединяет людей, широкая сеть
- **Конденсатор** — накапливает экспертизу и информацию
- **Проводник** — открывает доступ к закрытым кругам
- **Восходящая звезда** — быстро растёт, ценность будет расти
- **Инсайдер** — имеет доступ к информации раньше других
